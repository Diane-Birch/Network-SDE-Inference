The usage of Figure 3 code can follow the sequence provided below:

FOR SIMULATION:

vicsek_simulation.m
Simulation_flocks_Vicsek.ipynb

FOR EMPIRICAL SYSTEM:

empirical_data_process_nature.m (data augmentation and alignment)
Flock_dynamics_example.ipynb
manuscript_flocks_empirical_hf1_2dim.ipynb (hf2, hf3, and hf4)
Figure3.m (plot results)

Please note that the experimental data should be downloaded from https://doi.org/10.6084/m9.figshare.24804894.v4. Make sure to modify the file paths accordingly.
